
# ALTEC BRANDING | Precision Printing & Branding
ALTEC BRANDING is a high-quality custom printing and branding service provider. This repository contains the source code for the official business website, designed to showcase services, portfolios and streamline client inquiries for individuals, corporate clients, schools and events.

# 🚀 Live Demo
https://altecbranding.netlify.app/

# ✨ Features
Dynamic User Interface: Built with a modern, responsive design using custom CSS.<br>
Interactive Animations: Leverages GSAP (GreenSock Animation Platform) and ScrollTrigger for smooth, scroll-based animations and transitions.<br>
Interactive Mapping: Integrated Leaflet.js to provide location-based services and map views.<br>
Service Process Visualization: A dedicated section explaining the three-step "Consult, Design, Print" workflow.<br>
Mobile Optimized: Includes a fully functional mobile navigation overlay for seamless browsing on all devices.<br>
Brand Identity: High-resolution iconography using Font Awesome and a typography suite powered by Google Fonts (Montserrat & Open Sans).

# 🛠️ Tech Stack
HTML5: Semantic structure.<br>
CSS3: Custom styles including Flexbox/Grid layouts and animations.<br>
JavaScript: Logic for mobile navigation and interactive elements.<br>
GSAP & ScrollTrigger: Advanced UI/UX animations.<br>
Leaflet.js: Open-source interactive maps.<br>
Font Awesome: Vector icons.
